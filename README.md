# Emma-and-Alex-Wedding-Website
GitHub hosting for Emma and Alex's wedding website


## Photo uploads

The website now reads photo paths from `backend/photos/manifest.json`.

- Main photo: `images/photos/main.jpg`
- Gallery photos: `images/photos/small-1.jpg` to `images/photos/small-6.jpg`

Upload files to `images/photos/` with those names and they will appear automatically.
